# wanwan-html5
HTML5 version of the flash animation "WAN WAN INTENSIFIES.sfw". CSS animations controlled by Javascript.

# Why?
Wan not?

# Features
- Wolf girl
- Wan~
- Mobile support, including browsers unable to autoplay audio (Chrome for Android for example)
